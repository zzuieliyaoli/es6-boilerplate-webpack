# ES6 Boilerplate Webpack
A boilerplate for es6 demos.

## Usage

```
$ cd directory
$ npm install
```

Then, create `entry.js` wherever you want and modify `entry` in `./webpack.config.js`.

```
$ npm run start
```

## Features

- ES6 only, no React, Vue...
- Hot Module Replace
